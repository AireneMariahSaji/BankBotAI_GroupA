# BankBot AI – Chatbot for Banking FAQs

## Project Overview
BankBot AI is an intelligent chatbot designed to answer frequently asked banking queries.  
The chatbot uses a structured banking knowledge base stored in JSON format and provides quick responses to user questions.

This project was developed as part of the **Infosys Springboard 0.6 Program**.


## Features
- Banking FAQ chatbot
- JSON-based banking knowledge library
- Restricts responses to banking-related queries
- User-friendly chatbot interface


## Technologies Used
- Python
- Streamlit
- JSON
- Ollama (for AI integration)
